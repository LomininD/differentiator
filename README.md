# Differentiator

>*Данная версия readme промежуточная, более полное описание будет позже.*

Данная программа, анализирует математическое выражение рекурсивным спуском с дальнейшим преобразованием его в двоичное дерево. Функционал программы позволяет вычислить значение выражения, упростить его, посчитать n-ю производную. Результат работы представляется в виде сгенерированного pdf-документа.
